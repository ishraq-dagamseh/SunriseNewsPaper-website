# SunriseNewsPaper-website
In this project I create a website page (User Interface) for an imaginary Jordanian Newspaper, that I called it:"Sunrise Newspaper", to apply the basic concepts for html and css.

## IDE:
In this project I used Vscode IDE to write html code and live server app to run the code as a web page.
